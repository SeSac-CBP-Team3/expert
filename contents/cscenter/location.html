송파러닝센터

(주)엑스퍼트 아카데미 송파러닝센터

ADDRESS.

[05718] 서울특별시 중대로 121 롯데캐슬파인힐 2층

TEL.

02)2152-3915~4 / 3934 (평일 09:00~18:00)

E-MAIL.

open@exc.co.kr

지도 보기

자가용 이용 시

동부간선도로

동부간선도로(청담장지간) 수서IC 분당 방면으로 우측 고속도로 진입 2.8km 이동 → 동부간선도로(청담장지간) 남부순환로 수서IC 방면으로 우측 고속화도로 진출 2.1km 이동 → 양재대로 가락시장 방면으로 좌측도로 915m 이동 → 양재대로 방이역 길동사거리 방면으로 지하차도 진입 후 1.2km 이동 → 송이로 경찰병원 방면으로 우회전 후 611m 이동 → 송파대로28길 우회전 후 137m 이동 → 양재대로62길 좌회전 후 50m 이동 → 양재대로62길 우회전 후 42m 이동

버스 이용 시

경찰병원. 서울동부 고용노동지청정류장 하차

간선 301번, 간선 401번, 지선 3319번, 지선 3416번 경찰병원. 서울동부 고용노동지청정류장 하차 후 182m 이동

지하철 이용 시

경찰병원역 하차

1번 출구로 나와서, 서울 송파구 중대로 121까지 도보로 351m 이동

가락시장역 하차

4번 출구로 나와서, 서울 송파구 중대로 121까지 도보로 410m 이동
